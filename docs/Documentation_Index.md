## 🔑 KEY CONCEPTS EXPLAINED

### A&O Club Membership
- **What:** Free membership for guests
- **Benefits:** 25% discount on all bookings + loyalty points
- **Why:** Drive repeat bookings, increase customer lifetime value

### Room Types (7 total)
- Single Room (1 guest)
- Double/Twin Room (2 guests)
- Triple Room (3 guests)
- Family Room (4-8 guests, 1 double + multiple bunk beds)
- Mixed Dorm (6-8 guests per room)
- Female Dorm (6-8 female guests)
- Backpacker Dorm (budget option)

### Extras/Add-ons
- Towel rental (€2-€6)
- Breakfast (variable pricing)
- Parking (varies by location)
- Bicycle rental
- Late check-out
- Early check-in

### User Roles (5 types)
1. **Guest** - Book, review, earn loyalty points
2. **Reception Staff** - Check-in/check-out, services
3. **Property Manager** - Pricing, inventory, analytics
4. **Admin** - System configuration, multi-property
5. **SuperAdmin** - Full system control
